Resource Type: file_concat

Defined in:

Overview

Gets all the file fragments and puts these into the target file. This will mostly be used with exported resources.

example: File_fragment «| tag == ‘unique_tag’ |»

file_concat { ‘/tmp/file: tag => ‘unique_tag’, # Mandatory path => ‘/tmp/file’, # Optional. If given it overrides the resource name owner => ‘root’, # Optional. Default to undef group => ‘root’, # Optional. Default to undef mode => ‘0644’, # Optional. Default to undef order => ‘numeric’, # Optional, Default to ‘numeric’ ensure_newline => false, # Optional, Defaults to false }
