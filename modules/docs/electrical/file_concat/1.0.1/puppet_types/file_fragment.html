Overview

Create a file fragment to be used by file_concat. the file_fragment type creates a file fragment to be collected by file_concat based on the tag. The example is based on exported resources.

Example: @@file_fragment { "uniqe_name_$fqdn": tag => ‘unique_name’, order => 10, # Optional. Default to 10 content => ‘some content’ # OR content => template(‘template.erb’) # OR source => ‘puppet:///path/to/file’ }

Parameters

    Content

  • name (namevar)

    Unique name

  • order (defaults to: 10)

    Order

  • source

    Source

  • tag

    Tag name to be used by file_concat to collect all file_fragments by tag name
