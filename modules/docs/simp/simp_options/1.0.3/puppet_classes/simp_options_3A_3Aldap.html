Overview

simp_options::ldap class

Sets up LDAP configuration variables

Some parameters have default values in simp_options/data/ldap.yaml, all others must have a value specified via Hiera or your ENC.

Parameters:

  • base_dn (String) (defaults to: simplib::ldap::domain_to_dn()) —

    The Base Distinguished Name of the LDAP server

  • bind_dn (String) (defaults to: "cn=hostAuth,ou=Hosts,${simp_options::ldap::base_dn}") —

    The LDAP Bind Distinguished Name

  • bind_pw (String) —

    The LDAP Bind password

  • bind_hash (String) —

    The salted LDAP Bind password hash

  • sync_dn (String) (defaults to: "cn=LDAPSync,ou=Hosts,${simp_options::ldap::base_dn}") —

    The LDAP Sync Distinguished Name

  • sync_pw (String) —

    The LDAP Sync password

  • sync_hash (String) —

    The LDAP Sync password hash

  • root_dn (String) (defaults to: "cn=LDAPAdmin,ou=People,${simp_options::ldap::base_dn}") —

    The LDAP Root Distinguished Name

  • master (Simplib::URI) (defaults to: "ldap://${simp_options::puppet::server}") —

    The LDAP master in URI form (ldap://server)

  • uri (Array[Simplib::URI]) (defaults to: ["ldap://${simp_options::puppet::server}"]) —

    The list of OpenLDAP servers in URI form (ldap://server)

class simp_options::ldap (
  String              $bind_pw,
  String              $bind_hash,
  String              $sync_pw,
  String              $sync_hash,
  String              $base_dn    = simplib::ldap::domain_to_dn(),
  String              $bind_dn    = "cn=hostAuth,ou=Hosts,${simp_options::ldap::base_dn}",
  String              $sync_dn    = "cn=LDAPSync,ou=Hosts,${simp_options::ldap::base_dn}",
  String              $root_dn    = "cn=LDAPAdmin,ou=People,${simp_options::ldap::base_dn}",
  Simplib::URI        $master     = "ldap://${simp_options::puppet::server}",
  Array[Simplib::URI] $uri        = ["ldap://${simp_options::puppet::server}"]
){
  assert_private()
  validate_uri_list($master,['ldap','ldaps'])
  validate_uri_list($uri,['ldap','ldaps'])
}
