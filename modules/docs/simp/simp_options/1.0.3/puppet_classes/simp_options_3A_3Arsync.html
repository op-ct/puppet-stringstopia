Sets up rsync configuration variables

Parameters:

  • server (Simplib::Host) (defaults to: '127.0.0.1') —

    Server to use for the simp::rsync_server class.

    This class is used by several SIMP modules to efficiently synchronize large data files using rsync over stunnel.

    The default value of 127.0.0.1 is appropriate for rsync over stunnel.

  • timeout (Integer) (defaults to: 1) —

    rsync connection timeout in seconds
