Resource Type: simpcat_build

/tmp/tmp_git_repo20171214-12245-a2w2ff/lib/puppet/type/simpcat_build.rb
Providers:
simpcat_build

Overview

Build file from SIMP concat fragments

Properties

  • order (defaults to: ["*"])

    Array containing ordering info for build

Parameters

  • clean_comments

    If a line begins with the specified string it will not be printed in the output file.

  • clean_whitespace (defaults to: [:none])

    Cleans whitespace. Can be passed an array. ‘lines’ will cause the output to not contain any blank lines. ‘all’ is equivalent to [leading, trailing, lines]

    Supported values:
    • leading
    • trailing
    • lines
    • all
    • none
  • file_delimiter (defaults to: \n)

    Acts as the delimiter between concatenated file fragments. For instance, if you have two files with contents ‘foo’ and ‘bar’, the result with a file_delimiter of ‘:’ will be a file containing ‘foo:bar’.

  • name (namevar)
  • onlyif

    Copy file to target only if this command exits with status ‘0’

  • parent_build

    Specify the parent to this build step. Only needed for multiple staged builds. Can be an array. It does not make sense to specify a parent build without setting the target to the parent’s fragment directory.

  • quiet (defaults to: false)

    Suppress errors when no fragments exist for build

    Supported values:
    • true
    • false
  • sort (defaults to: false)

    Sort the built file. This tries to sort in a human fashion with 1 < 2 < 10 < 20 < a, etc.. sort. Note that this will need to read the entire file into memory

          Example Sort:

      ['a','1','b','10','2','20','Z','A']

      translates to

      ['1','2','10','20','a','A','b','Z']

      Note: If you use a file delimiter with this, it *will not* be sorted!
    Supported values:
    • true
    • false
  • squeeze_blank (defaults to: false)

    Never output more than one blank line

    Supported values:
    • true
    • false
  • target

    Fully qualified path to copy output file to

  • unique (defaults to: false)

    Only print unique lines to the output file. Sort takes precedence. This does not affect file delimiters.

    true: Uses Ruby’s Array.uniq function. It will remove all duplicates regardless of where they are in the file.

    uniq: Acts like the uniq command found in GNU coreutils and only removes consecutive duplicates.

    Supported values:
    • true
    • false
    • uniq
