simplib_deprecation() ⇒ Nil

Function to print deprecation warnings for 3.X functions. The first argument is the uniqueness key, which allows deduping of messages. The second argument is the message to be printed. Messages can be suppressed if the SIMPLIB_LOG_DEPRECATIONS environment variable is set to ‘false’.
