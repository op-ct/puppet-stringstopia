Overview

Set up the /etc/login.defs configuration file. All option values are taken directly from the system documentation.

Any parameter that is a list will require an array to be passed.

NOTE: pass_min_len and pass_max_len will NOT have any effect on a stock RedHat machine. * Max length will only affect 3des encryption, which is not used on modern machines. * Min length should be configured using /etc/pam.d/ or /etc/security/pwquality.conf.

author: SIMP Team simp@simp-project.com
